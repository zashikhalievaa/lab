## Слайд 3: Настройка DNS-сервера по умолчанию

**Команды nmcli:**
```bash
nmcli> remove ipv4.dns
nmcli> set ipv4.ignore-auto-dns yes
nmcli> set ipv4.dns 127.0.0.1
nmcli> save
Результат:

DNS-сервер настроен на localhost (127.0.0.1)

Отключено автоматическое получение DNS

![Установка пакетов](images/3.jpg)

Слайд 4: Настройка направлений DNS-запросов
